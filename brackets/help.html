General Rules

BRACKETS has two type of levels. These levels can be solved with the following ciphers(tables) by replacing the alphabets with numbers -

  • Latitude and Longitude level

    In this level the latitude and longitude of the city must be calculated from a set of two codes.The codes generate numeric values which can be used to locate a city in the map provided.

  • Location level

    In this level name of a location in the city must be found out based on code provided in that level.

  • There is no negative marking till level 20. After that only three atempts can be made at one question after that negative marking will be there
  • Five levels can be bypassed.

Winners will be decided by clause of ordering by level then by points then by who reached that level first.

Guide to City Level

The latitude and longitude of a city is used to locate it in a map. The latitude and longitude can be either positive or negative.

  • All the expressions are evaluated by selecting first two values (operands) and then applying the first operator encountered in left to right manner.
  • AS implies a decimal point, the code to the left and that to the right of the decimal have no effect on each other.
  • If a negative sign remains in the result that means the answer is negative.
  • If latitude is positive it implies north, if it is negative it implies south.
  • If longitude is positive it implies east, if it is negative it implies west.
